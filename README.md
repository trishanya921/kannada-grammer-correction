# Kannada Grammar Corrector 🎯

A beautiful web application that corrects grammatical errors in Kannada sentences using Google's Gemini AI.

## Features

- ✨ Real-time Kannada grammar correction
- 🎨 Modern and responsive UI design
- 📝 Shows changes made to the sentence
- 📋 Copy corrected text with one click
- ⌨️ Keyboard shortcut support (Ctrl/Cmd + Enter)
- 🌐 Works entirely in the browser
- 💜 Beautiful gradient design with smooth animations

## How to Use

1. Open `index.html` in your web browser
2. Type or paste a Kannada sentence with grammatical errors
3. Click "Correct Grammar" button or press Ctrl/Cmd + Enter
4. View the corrected sentence and changes made
5. Copy the corrected text if needed

## Technologies Used

- HTML5
- CSS3 (with modern gradients and animations)
- JavaScript (Vanilla JS)
- Google Gemini AI API

## API Key

The Google API key is already configured in the `script.js` file. If you need to change it, edit the `API_KEY` constant at the top of `script.js`.

## Browser Compatibility

Works on all modern browsers:
- Chrome
- Firefox
- Safari
- Edge

## Note

- Make sure you have an active internet connection to use the AI features
- The application supports Kannada Unicode text (Noto Sans Kannada font recommended)

## Privacy

- All processing is done through Google's Gemini API
- No data is stored locally or on any server
- Each session is independent
